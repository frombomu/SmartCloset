# SmartCloset
Smart Closet is a simple web app that helps users choose clothes based on the weather.
Users enter a city or province name, and the app shows the weather and outfit suggestions using images.

Features
- Search weather by city or province name
- Show temperature in Celsius
- Show weather description
- Suggest outfits based on temperature
- Show an umbrella reminder when it is rainy

Built With
HTML – page structure and page design
CSS – image size
JavaScript – logic and API request
OpenWeather API – weather data
